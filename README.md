# Challenge Adcap

## Instrucciones

### Instalar las dependencias
```sh
npm i && npm install styled-components @types/styled-components --legacy-peer-deps
```

### Instalar EAS (expo notifications)
```sh
npm install --global eas-cli
```

### Loguearse con Expo
```sh
eas login
```

### Configurar la build
```sh
eas build:configure
```

### Levantar proyecto
```sh
npm start
```
